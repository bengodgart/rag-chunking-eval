How to read this

hit-rate@5 is the fraction of questions whose gold chunk (the right document and, when the question carries an answer span, a chunk that actually contains that span) landed in the top 5 retrieved chunks. MRR@5 is the mean of 1/rank of the first relevant chunk, counted 0 when none was in the top 5. Retrieval is a hand-rolled BM25 index in memory. No embedding API, no vector database, no network call.

Generated by chunkeval. Offline, deterministic, standard library only.
